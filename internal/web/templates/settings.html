Memory & distillation

Each day the assistant reads the day's conversation and saves a dated memory note (it also refreshes it every few hours). With notifications on, it sends you that note when the end-of-day pass runs, so you can see what it remembered. See the Guide for how memory works.

Context assembly

How much context each turn carries: the last N messages of this thread, plus the top K relevant memories retrieved by search (the last two days of notes are always loaded regardless of K).
